.📑 Informe de Proyecto: Infraestructura Digital Climair S.L.Este proyecto integra la administración de servidores Linux con la gestión avanzada de estructuras de datos para optimizar los reportes de intervenciones técnicas.🛡️ 1. Infraestructura de Servidor (Módulo ISO)Se ha desplegado un servidor Ubuntu Server 24.04 LTS que actúa como el núcleo de datos de la empresa.Almacenamiento: Implementado en la ruta /srv/climair/ con una estructura de permisos estricta para el grupo tecnicos.Interoperabilidad (Samba): Se ha configurado un servidor de archivos Samba para permitir que equipos Windows (administrativos) gestionen los archivos XML y JSON de forma nativa.Seguridad: Control de acceso mediante usuarios de red y configuración de UFW (Firewall) para proteger los datos de la empresa.📊 2. Gestión de Información (Módulo LMG)La digitalización de los partes de trabajo se ha realizado bajo estándares modernos de intercambio de información.Estructura XML: Un archivo datos.xml centraliza los datos de técnicos y sus intervenciones.Validación DTD: Se ha creado un contrato de datos (modelo.dtd) que asegura que ninguna intervención se guarde con errores o estados no permitidos, garantizando la integridad de la información.Visualización XSLT/CSS: Para la gerencia, los datos en bruto se transforman en un Dashboard Futurista (estilo neón/glassmorphism) que permite monitorizar el estado de los equipos de un vistazo.Exportación JSON: Se provee un archivo datos.json para la futura integración con aplicaciones móviles de los técnicos.🔗 3.
